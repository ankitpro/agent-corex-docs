## Documentation Purpose

Agent-CoreX is a dynamic tool retrieval framework for AI agents using the Model Context Protocol (MCP). This documentation helps developers understand and implement the system.

## Key Concepts to Emphasize

1. **Dynamic Tool Retrieval** - Finding tools using natural language, not hardcoding
2. **MCP Protocol** - Standard interface for AI-tool communication
3. **Tool Ranking** - Intelligent selection based on relevance, popularity, history, performance
4. **Real-World Use Cases** - GitHub, DevOps, Jira+Slack, Infrastructure automation

## FAQ Answers

**Q: What is Agent-CoreX?**
A: Framework for AI agents to dynamically discover and execute tools via MCP protocol.

**Q: How is it different?**
A: Instead of hardcoding tools, agents query for relevant ones at runtime.

**Q: Which MCP servers are supported?**
A: 100+ servers including GitHub, Slack, Jira, Terraform, Docker, AWS, Kubernetes, etc.

**Q: Can I use it without MCP?**
A: No, MCP is the core protocol. But custom servers are easy to build.

**Q: Is it production-ready?**
A: Yes, designed for enterprise use with security, rate limiting, monitoring.
