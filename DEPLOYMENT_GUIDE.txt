DEPLOYMENT GUIDE: Agent-CoreX Docs to Mintlify

=== QUICK DEPLOYMENT (5 minutes) ===

Option 1: Mintlify Cloud (RECOMMENDED)
1. Go to https://mintlify.com
2. Click "Start Building"
3. Sign in with GitHub
4. Select agent-corex-docs repository
5. Click "Deploy"
6. Configure custom domain: docs.agent-corex.com

Option 2: Vercel
1. Install Vercel CLI: npm install -g vercel
2. Run: vercel
3. Connect to GitHub repo

Option 3: Netlify
1. npm install -g netlify-cli
2. mintlify build
3. netlify deploy --prod --dir=./site

=== LOCAL DEVELOPMENT ===

Install Mintlify:
npm install -g mintlify

Run locally:
mintlify dev
# Visit http://localhost:3000

Build static:
mintlify build
# Output in ./site/ directory

=== CUSTOMIZATION ===

Before deployment, update:
1. mint.json - Add your logo, colors, links
2. Logo files - Add dark.svg and light.svg to public/logo/
3. Links - Update signup URL, GitHub, Slack
4. Favicon - Add public/favicon.ico
