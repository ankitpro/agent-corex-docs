# Agent-CoreX Documentation

Complete Mintlify documentation system for Agent-CoreX MCP framework.

## Overview

Agent-CoreX is a **dynamic tool retrieval framework** for AI agents using the Model Context Protocol (MCP). This documentation covers:

- Getting started with Agent-CoreX
- Core concepts (tool lifecycle, dynamic retrieval, ranking)
- MCP servers for developers
- Tool retrieval system
- Complete API reference
- Real-world use cases and examples

## Quick Start

### Local Development

```bash
# Install Mintlify
npm install -g mintlify

# Run documentation locally
mintlify dev

# Open browser to http://localhost:3000
```

### Deploy to Mintlify

1. Push to GitHub
2. Go to mintlify.com
3. Connect GitHub repository
4. Auto-deploy on commits
